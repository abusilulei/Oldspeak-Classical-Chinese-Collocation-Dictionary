# Oldspeak-Classical-Chinese-Collocation-Dictionary
Author: Yifei Liu
Date: 2025-05-01

## Overview:
Oldspeak is a collocation dictionary for classical chinese, built with a large corpus of ancient Chinese texts and modern NLP techniques.

You can search vocabulary, collocations, and linguistic patterns from ancient texts. 

It is perfect for students, scholars, and language enthusiasts eager to master the nuances of classical Chinese writing.
